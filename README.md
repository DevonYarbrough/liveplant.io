# liveplant.io

Liveplant is an experiment where we put the life of a plant in the hands of the internet.

### To Install:

**Requires io.js 2+**

***or you can transpile to ES5 with Babel to use the cli tool with node.js... io.js is probably better***

```
npm i -g seeds
npm install
bower install
seeds serve
```
